## **5-Minute Setup**

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Set Gemini API key (PowerShell)
$env:GEMINI_API_KEY="your_api_key_here"

# 3. Launch app
streamlit run app.py
```
